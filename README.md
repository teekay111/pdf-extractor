# AI PDF Extractor

A native Mac application that uses Google Gemini to extract structured data from PDF documents.

## Features
- **Drag & Drop Interface**: Easily upload multiple PDFs.
- **AI-Powered Extraction**: Uses Gemini Flash for fast and accurate data extraction.
- **Custom Schema**: Define exactly what fields you want to extract.
- **Native Experience**: Runs as a standalone Mac app.
- **Privacy Focused**: Data is processed via your API key and saved locally to your machine.
